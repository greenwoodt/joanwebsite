# joanazpetita.com 

A simple professional portfolio built on Rails 8. The webpage uses some Javascript to automatically play whilst loaded on to the site, allowing you to see a sneak preview of Joan's work. This also works while in a responsive view on mobile utilising a different video made for mobile.
